# Mandelbrot

Mandelbrot set generator in C / python.

Just for fun!

### Requirements
- C version: [libpng](http://www.libpng.org/pub/png/libpng.html)
- Python version: [Pillow](https://python-pillow.org/)
  - On python 2.X: Run `.py` file.
  - On python 3.X: Run `.py3` file.
